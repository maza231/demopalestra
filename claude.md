# Project Rules

This project is a modern website for a small gym.

## Tech stack
- React / Next.js
- Tailwind CSS
- Mobile-first design
- Clean component structure

## Design style
- Modern, premium, simple
- Strong hero section
- Clear CTA buttons
- Good spacing
- No generic AI-looking layout
- No random gradients unless useful

## Code rules
- Use reusable components
- Keep code readable
- Avoid unnecessary dependencies
- Make all pages responsive
- Optimize images
- Test layout on mobile

## Business goal
The website should generate more customers for this gym through a message on WhatsApp or a call to the gym's phone number or  even through an in-person visit to the gym.
